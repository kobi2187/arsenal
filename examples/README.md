# Arsenal Examples

This directory contains examples demonstrating how to use Arsenal.

## Building and Running

```bash
# Build an example
nim c examples/basic_usage.nim

# Run it
./basic_usage
```

## Examples

- `basic_usage.nim` - Basic platform detection and strategies
- `echo_server/` - Concurrent echo server (when implemented)
- `lock_free_queue/` - Lock-free queue usage (when implemented)
